Tanıtım
Bu proje, ASP.NET Entity Framework kullanılarak geliştirilmiş bir web uygulamasıdır.
Projede, MSSQL Server veritabanı kullanılmıştır.

Projede kullanılan yapılar :
MSSQL Server
ASP .NET Entity Framework
Crud İşlemleri
LinQ Expression
